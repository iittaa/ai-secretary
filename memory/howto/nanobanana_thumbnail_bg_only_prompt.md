# Gemini nano banana サムネ【背景＋テキストのみ生成用】

**コンポーザースクリプトとセット**で使うプロンプト。
キャラなしで背景＋テキストだけ生成 → 後処理でキャラを固定サイズ合成する流れ。

## ワークフロー

1. **このプロンプト**でGeminiから「背景＋テキストのみ」のサムネを生成
2. キャラ画像（透過PNG推奨）を用意
3. **`tools/thumbnail_composer.py`** で右下にキャラを固定サイズ合成

```bash
python3 tools/thumbnail_composer.py base.png char.png -o final.png -s 0.08 -m 0.10
```

## メリット
- ✅ キャラのサイズが**完全固定**（Geminiの解釈に左右されない）
- ✅ キャラの位置がピクセル単位で正確
- ✅ キャラ画像を変えても結果が安定
- ✅ 量産が高速（背景＋テキストは1回作れば使い回し可能）

## 使い分け

| 目的 | 使うファイル |
|---|---|
| キャラ込みで一発生成 | `nanobanana_thumbnail_prompt` |
| **背景＋テキストのみ生成（合成前提）** | **このファイル** |
| 完成サムネのフォント変更 | `nanobanana_thumbnail_temp_prompt`（用途による） |
| 完成サムネのキャラ差し替え | `nanobanana_thumbnail_remix_prompt` |

## ポイント
- 右下エリア（画面幅70〜100% × 高さ70〜100%）を**空ける指示**を入れている
- キャラ後乗せ用の余白を背景のまま残す
- 装飾やテキストもこのエリアには配置しない

## プロンプト本体

```
画像1を背景として使用し、特典告知用サムネイル画像（テキスト入り・キャラクターなし）を生成してください。
※この画像は後処理でキャラクターアイコンを右下に合成するため、右下エリアを空けておくこと。

【レイアウト（最重要・絶対厳守）】
- テキストは画面左側エリアに配置（画面幅 0〜65% の範囲内）
- 画面右下エリア（画面幅 70〜100%、高さ 70〜100%）は何も描画しない
  - キャラクター・装飾・模様・テキストなど一切配置しない
  - 背景の柄はそのままでOK（後でキャラを合成する余白として確保）
- 画面右下エリアにはテキストを絶対にはみ出させない

【背景】
- 画像1をそのまま使用。改変・再描画しない

【テキスト】
- 文言（鉤括弧「」『』""''（）[]などの記号は描画しない）：
  - 1行目：月収30万円
  - 2行目：マネタイズロードマップ
- 揃え：左揃え（中央揃え・右揃え禁止）。両行の左端X座標を完全一致（画面左端から約5〜7%の位置）
- 位置：テキストエリア（画面幅0〜65%）の上下中央
- サイズ：エリア内に収まる範囲で読みやすく目立つ大きさ。1行目と2行目のサイズ差は1.5倍以内
- デザイン統一：書体・太さ・文字色・フチ・影は両行で完全一致
- 書体：高級感のある明朝体／セリフ書体（ゴージャス感、やや太め）
- 文字色：メタリックゴールドの縦グラデ（上部:明るいシャンパンゴールド／中央:鮮やかなゴールド／下部:ディープゴールドorブロンズ）
- 光沢：文字上部に細いハイライト線を入れて金属光沢・艶を演出
- フチ：ダークブラウンまたは黒の細フチ
- 影：淡いドロップシャドウ
- イメージ：ジュエリーブランドのロゴのような艶やかで立体的な金文字
- Both lines: left-aligned, identical left x-coordinate, size within 1.5x ratio, identical typeface/color/outline/shadow.

【禁止事項】
- キャラクターを一切描画しない（人物・アイコン・イラスト含む）
- 装飾要素の追加禁止：縦線/横線/罫線/バー/ゴールドライン/区切り線/枠/フレーム/ボーダー/追加アイコン/エンブレム/スター等
- 右下エリア（画面幅70〜100%、高さ70〜100%）には何も新規追加しない
- 日本語の文字化け禁止（一字一句正確に描画）

【出力】
- アスペクト比：16:9
- 全体トーン：上質・高級感・ゴージャス（ネオン・原色は使わない）

Generate a thumbnail with text only, NO character at all. The bottom-right area (70-100% width, 70-100% height) MUST be left empty (just background) for post-processing character compositing.
```
