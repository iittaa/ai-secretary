# Gemini nano banana サムネ生成プロンプト【ベース】

背景画像＋キャラ画像＋テキストで特典サムネを作るときの**ベースプロンプト**。
今後nanobananaでサムネ生成するときは、これを起点に調整する。

## 使い方
- 画像1に背景、画像2にキャラを添付
- テキスト部分の文言を差し替えるだけで量産可能
- 文字化けしたら文言を短く or Canvaで後乗せ

## プロンプト本体

```
画像1を背景、画像2をキャラクターとして使用し、特典告知用サムネイル画像を生成してください。

【合成指示】
- 背景：画像1をそのまま使用。改変・再描画しない
- キャラ：画像2の人物を右下に配置。顔・髪・服・画風は画像2と完全同一（再描画禁止）
- キャラサイズ：画面の約8%
- 【厳守】キャラの右端と画面右端の間、下端と画面下端の間に、それぞれ画面幅・高さの10〜18%の余白を確保（端に接触・はみ出し禁止、中央寄りすぎもNG）
- 背景とキャラの境界は自然に馴染ませる（軽い影はOK）

【テキスト】
- 文言（鉤括弧「」『』""''（）[]などの記号は描画しない）：
  - 1行目：月収30万円
  - 2行目：マネタイズロードマップ
- 配置エリア：画面左端〜画面幅の65%まで（キャラ領域と完全に重ならないよう厳守）
- テキストの右端は画面幅の65%を絶対に超えないこと（超えそうならサイズを縮小する）
- 揃え：左揃え（中央揃え・右揃えは禁止）。両行の左端X座標を完全一致（画面左端から約5〜7%の位置）
- 位置：配置エリアの上下中央
- サイズ：各行が配置エリア幅の80%以内に収まる大きさ。行ごとに自動調整してよいが、1行目と2行目のサイズ差は1.5倍以内
- デザイン統一：書体・太さ・文字色・フチ・影は両行で完全一致
- スタイル：高級感のある明朝体／セリフ書体（ゴージャス感、やや太め）
- 文字色：メタリックゴールドの縦グラデーション（上部：明るいシャンパンゴールド／中央：鮮やかなゴールド／下部：ディープゴールドまたはブロンズ）
- 光沢表現：文字上部に細いハイライト線を入れて金属光沢・艶を演出
- フチ：ダークブラウンまたは黒の細フチ（淡い背景でも視認性を確保）
- 影：淡いドロップシャドウで軽く浮かせる
- 全体イメージ：ジュエリーブランドのロゴのような、艶やかで立体的な金文字
- Both lines: left-aligned, same left x-coordinate, size within 1.5x ratio, identical typeface/color/outline/shadow.

【禁止事項】
- 【最重要】キャラとテキストの重なり絶対禁止。キャラの外形とテキストのバウンディングボックスの間に最低でも画面幅の5%の余白を確保
- 【最重要】装飾要素の追加は一切禁止。特にテキストの左側・右側・上側・下側への縦線/横線/罫線/バー/ゴールドライン/区切り線の描画を厳禁（No vertical lines, no horizontal lines, no decorative bars, no dividers, no gold stripes anywhere around the text）
- 枠・フレーム・ボーダー・囲み装飾・追加アイコン・エンブレム・スター類の追加も禁止
- テキストは背景の上にそのまま配置するのみ。テキスト周辺に追加のグラフィック要素を一切描かないこと
- 日本語の文字化け禁止（一字一句正確に描画）

【出力】
- アスペクト比：16:9
- 全体トーン：上質・高級感重視（ネオン・原色は使わない）
```

## コツ
- キャラが別人化したら「画像2のキャラクター性を100%維持、顔を再描画しない」を追加
- 配置・サイズは具体的な数値で指定すると精度上がる
- 日本語テキストがうまくいかない場合：文言を短く、またはCanvaで後乗せ
