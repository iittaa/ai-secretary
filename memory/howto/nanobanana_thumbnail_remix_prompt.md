# Gemini nano banana サムネ量産プロンプト【リミックス用】

完成したベースサムネをテンプレートとして、**文字とキャラだけ差し替えて量産**するためのプロンプト。

## 使い方
1. まずベースとなる完璧な1枚を作成（`nanobanana_thumbnail_prompt.txt`を使用）
2. 量産時はこのプロンプトを使用：
   - **画像1**：完成済みベースサムネ
   - **画像2**：差し替えたいキャラ画像
   - プロンプト内の「1行目」「2行目」を新しい文言に書き換え

## メリット
- ✅ サイズ・位置・デザインがほぼ固定される
- ✅ デザインの統一感が自動でキープ
- ✅ プロンプトが短く済む
- ✅ 量産が高速

## 注意点
- たまに微妙にレイアウトずれる（その時は再生成）
- キャラの顔が変わるリスクは残る → 「画像2の顔を完全維持」を強調
- 完璧な再現性が必要なら、後処理（PIL/Canva）で合成

## プロンプト本体

```
画像1（ベースとなるサムネ画像）のレイアウト・デザイン・スタイル・配色を完全に維持し、以下の要素だけを差し替えて新しいサムネを生成してください。

【ベース画像】
- 画像1：完成済みのサムネ。これを「テンプレート」として扱い、構図・配置・フォント・色・装飾を一切変えない

【差し替え対象】
- テキスト：
  - 1行目：月収30万円
  - 2行目：マネタイズロードマップ
  - ※ 鉤括弧「」『』""''などの記号は描画しない
  - ※ 文言だけ差し替え、フォント・色・サイズ・位置はベース画像と完全に同一に保つ
- キャラクター：
  - 画像2の人物を使用。顔・髪・服・画風は画像2と完全同一（再描画禁止）
  - 配置位置・サイズ・余白はベース画像のキャラと完全に同じにする

【絶対に変更しないもの】
- 背景の柄・色・装飾
- テキストのフォント・色・グラデ・フチ・影・光沢
- テキストの位置・揃え・行間
- キャラの配置位置・サイズ
- 全体のアスペクト比・トーン

【禁止事項】
- 装飾要素の追加禁止：縦線/横線/罫線/バー/枠/フレーム/追加アイコン等
- ベース画像にない要素を一切追加しないこと
- 日本語の文字化け禁止（一字一句正確に描画）
```
