# Gemini nano banana サムネ【一時プロンプト・スクラッチパッド】

その時々の用途に応じて中身を書き換える**一時ファイル**。
固定用途ではなく、毎回使い捨てで書き換えてOK。

## 現在の用途
**右下のキャラクターアイコンを0.8倍に縮小する**プロンプト

## 使い方
1. ベースサムネを画像1として添付
2. 生成 → キャラだけ0.8倍に縮小、他は全て同じ

## 過去の用途履歴
- フォントデザインだけ差し替え（メタリックゴールド明朝）
- キャラクターサイズだけ変更
- キャラクターアイコンだけ差し替え
- キャラを0.8倍に縮小 ← 現在

## プロンプト本体（現在版）

```
画像1（ベースサムネ）の文言・背景・テキストデザイン・キャラクターをすべて維持し、右下のキャラクターアイコンのサイズだけを現在の0.8倍に縮小してください。

【ベース画像】
- 画像1：完成済みのサムネ

【維持するもの（変更しない）】
- テキストの文言・書体・色・フチ・影・光沢・サイズ・位置・揃え
- 背景画像（柄・色・装飾すべて）
- キャラの顔・髪・服・ポーズ・表情・画風（再描画禁止、見た目はそのまま）
- キャラの右下角アンカー位置（右端から内側10%、下端から内側10%）
- 全体のアスペクト比（16:9）
- 全体のレイアウト構成

【変更対象（サイズのみ）】
- 右下のキャラクターアイコンのサイズを現在の0.8倍に縮小
- 縮小後も右下角アンカーを維持（右端・下端から10%の位置に固定）
- 縮小により生まれる余白は背景がそのまま見えるようにする

【禁止事項】
- 装飾要素の追加禁止：縦線/横線/罫線/バー/枠/フレーム/追加アイコン等
- ベース画像にない要素を一切追加しないこと
- テキスト・背景・キャラの見た目を変えない（サイズだけ）

Keep all elements from image1 exactly the same. Only resize the character icon at the bottom-right to 0.8x of its current size, while keeping it anchored at the bottom-right corner with the same 10% inset margins.
```
